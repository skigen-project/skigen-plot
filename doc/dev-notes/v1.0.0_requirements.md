# SkigenPlot v1.0.0 — Release Requirements

## Mission

Provide the ubiquitous, low-friction plotting standard for the C++ scientific
and ML ecosystem.  Think *matplotlib for C++*, but hardware-accelerated from
day one.

## Scope (MVP)

### Supported Plot Types

| Type | Dimension | Input | Notes |
|------|-----------|-------|-------|
| Line plot | 2D | `VectorXf` × 2 (x, y) | Solid polyline, per-series colour |
| Scatter plot | 2D | `VectorXf` × 2 (x, y) | Instanced quads or point sprites |
| Scrolling telemetry | 2D | Ring buffer / streaming `VectorXf` | Fixed-width sliding window |
| Point cloud | 3D | `MatrixXf` (N × 3) | Per-vertex colour, depth test |
| Surface mesh | 3D | Vertices (N × 3) + indices (M × 3) | Flat or smooth shading |

### Public API Contract

* Accept `const Eigen::MatrixBase<Derived>&` on every data-facing entry point.
* Zero forced heap allocations on the plotting hot path when the caller passes
  a contiguous `Eigen::MatrixXf` or `Eigen::Map`.
* Bridge any `std::ranges::contiguous_range` of arithmetic type into an
  `Eigen::Map` via a single-call helper.

### Rendering Architecture

* All rendering through `QRhiWidget` (Qt ≥ 6.7).
* Backends: Vulkan (Linux/Android), Metal (macOS/iOS), Direct3D 12 (Windows).
* **No QPainter / Qt Canvas** — prohibited to keep the library LGPLv3/MIT.
* Shaders compiled to `.qsb` via `qt_add_shaders()` at build time.
* Public headers forward-declare all QRhi types; `#include <rhi/qrhi.h>` is
  confined to `.cpp` translation units only.

## Performance Targets

| Metric | Target |
|--------|--------|
| 2D line (1 M points) | ≥ 60 Hz sustained |
| 3D point cloud (500 K points) | ≥ 60 Hz sustained |
| 3D mesh (200 K triangles) | ≥ 60 Hz sustained |
| Data upload latency | < 2 ms for 1 M floats |
| First frame after `plot()` | < 16 ms |

## Dependencies

| Library | Version | Link | Exposure |
|---------|---------|------|----------|
| Eigen 3 | ≥ 3.4 | INTERFACE (header-only) | PUBLIC |
| Qt 6 Core/Gui/Widgets | ≥ 6.7 | Shared/static | PUBLIC (Widgets) |
| Qt 6 ShaderTools | ≥ 6.7 | Build-time only | PRIVATE |
| Qt 6 GuiPrivate | ≥ 6.7 | QRhi internals | PRIVATE |

## Language & Build

* C++23 (`std::span`, `std::expected`, `std::ranges`, `std::numbers`).
* Target-based CMake (≥ 3.20), exported as `SkigenPlot::SkigenPlot`.
* Strict warnings: `-Wall -Wextra -Wpedantic` (GCC/Clang), `/W4` (MSVC).

## License Verification Criteria

1. No translation unit includes a GPL-only Qt module header.
2. No `QPainter`, `QGraphicsScene`, or `QOpenGLWidget` usage anywhere.
3. All rendering passes through `QRhi` accessed via `QRhiWidget` (LGPLv3).
4. Eigen is MPL2 — compatible with MIT.
5. Final deliverable is MIT-licensed.
